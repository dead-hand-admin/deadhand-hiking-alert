# DeadHand - Hiking Safety Alert System

Система "мёртвой руки" для туристов. Автоматическое оповещение спасательных служб при отсутствии связи в контрольных точках.

## Особенности

- 🏔️ Многоэтапные походы с контрольными точками
- ⏰ Автоматический мониторинг дедлайнов
- 📧 Оповещение контактов и МЧС
- 🌍 Поддержка нескольких стран и служб спасения
- 🌐 Мультиязычность (RU/EN)

## Технологии

- PHP 8.3 (MVC)
- MySQL/MariaDB
- Tailwind CSS + Alpine.js
- Mailgun API

## Структура
```
/htdocs/

│   └── Helpers/
├── templates/
├── lang/
├── migrations/
└── uploads/
```

## Установка

1. Клонируйте репозиторий
2. Скопируйте `.env.example` в `.env`
3. Настройте базу данных
4. Примените миграции: `php migrations/run.php?key=YOUR_KEY`
5. Настройте веб-сервер на `/htdocs`
